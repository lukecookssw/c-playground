# C Keywords
| Keyword   | Description                            |
|-----------|----------------------------------------|
| auto      | declares automatic variables           |
| break     | breaks out of loops                     |
| case      | selects code block in switch statement |
| char      | declares character variables            |
| const     | declares read-only variables            |
| continue  | skips to next iteration of loop         |
| default   | default code block in switch statement  |
| do        | loops while a condition is true         |
| double    | declares double-precision variables    |
| else      | executes if previous conditions fail   |
| enum      | declares enumerated types               |
| extern    | declares variables in another file      |
| float     | declares floating-point variables       |
| for       | loops a specific number of times        |
| goto      | jumps to a labeled statement            |
| if        | executes code if a condition is true    |
| int       | declares integer variables              |
| long      | declares long integer variables         |
| register  | declares register variables             |
| return    | returns a value from a function         |
| short     | declares short integer variables        |
| signed    | declares signed variables               |
| sizeof    | returns size of a variable in bytes     |
| static    | declares static variables               |
| struct    | declares a structure type               |
| switch    | selects code block based on a variable  |
| typedef   | creates a new type                      |
| union     | declares a union type                   |
| unsigned  | declares unsigned variables             |
| void      | declares functions with no return value |
| volatile  | declares volatile variables             |
| while     | loops while a condition is true         |
